102-fdo-gr1-no-pid!

This landing page has no FAIR Signposting HTTP headers, in particular it is missing the rel=cite-as PID indicator.

This is a negative test for FDO principle FDO-GR1: A PID, standing for a globally unique, persistent and resolvable identifier, is assumed to be the basis for FAIR Digital Objects. Every FDO is assigned one or more PIDs.

This test deliberately have a name with the ! character which is not valid as a PID https://w3id.org/a2a-fair-metrics/102-fdo-gr1-no-pid!/ (404 Not Found), so this test can be used both for a non-resolvable PID and for a URI which have no corresponding PID.
